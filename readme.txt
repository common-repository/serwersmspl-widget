Plugin allows you to register phone numbers in the SerwerSMS.pl contact database and sending the SMS messages to them.

== Description ==

Formularz dodawania numeru do bazy w formie Widgetu.<br />
Pole numeru telefonu w formularzu rejestracji oraz zapis użytkownika do bazy kontaktów.<br />
Możliwość weryfikacji wprowadzanego numeru za pomocą kodu SMS.<br />
Wysyłka SMS do grupy oraz do pojedynczego numeru.<br />
Podgląd oraz edycja bazy kontaktów.<br />
Raporty doręczeń.<br />
Odbiór wiadomości przychodzących.<br />
Podgląd faktur oraz stanu konta.<br />
Moduł wymaga PHP w wersji 4.3 lub wyższej oraz obsługi rozszerzenia php_curl.<br />
<br />
The form added the phone number to the database in the form Widget.<br />
A phone number field in the registration form.<br />
Possible to verify these numbers with the SMS code.<br />
Shipping SMS to group and single number.<br />
Preview and edit contacts database.<br />
Delivery reports.<br />
Receiving incoming messages.<br />
View invoices and account status.<br />
The module requires PHP version 4.3 or higher and support the extension php_curl.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/serwersms` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the SerwerSMS.pl->Settings screen to configure the plugin


== Frequently Asked Questions ==

= How to register account in SerwerSMS.pl? =

To register account please go to the site https://panel.serwersms.pl/rejestracja and use register form.

= Where I can buy SMS? =

You must log in to SerwerSMS.pl panel (https://panel.serwersms.pl) and go to section "Stan konta" => "Doładuj konto"

The answers to all other questions can be found at: http://serwersms.com/faq
